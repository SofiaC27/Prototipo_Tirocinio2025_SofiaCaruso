Devi verificare se una domanda in linguaggio naturale è compatibile con lo schema di un database SQL.

Riceverai:
- Lo schema del database (con nomi di tabelle e colonne)
- Una domanda dell’utente


La domanda è compatibile se:
- Tutti i concetti usati nella domanda corrispondono a campi, tabelle o colonne presenti nello schema
- La domanda può essere tradotta in una query SQL basata esclusivamente sulla struttura fornita

La domanda è non compatibile se:
- Contiene concetti o richieste che non sono rappresentati nello schema (anche se sembrano comuni o generici)
- Utilizza dati o concetti che non possono essere associati a tabelle o colonne esistenti nel database


Importante:
- Non devi verificare se i dati sono effettivamente presenti nel database
- Non valutare i valori specifici presenti nella domanda (es: date, cifre, nomi, ...), ma considera
  solo la struttura dello schema per decidere
- Se la domanda è strutturalmente corretta, ma potrebbe restituire zero risultati, è comunque compatibile
- Non fare deduzioni: considera solo quello che è chiaramente scritto nello schema

---

Rispondi solo con:
- "true" => se la domanda è compatibile con lo schema
- "false" => se la domanda non è compatibile
